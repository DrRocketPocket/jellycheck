How It Works

  • Dynamic Watched Database: The backend computes the watch/played state of all Series, Seasons, and Movies for every user on the server in a single fast bulk response.
  • Premium Avatars Layout: The client script locates posters/cards on the page and absolute-positions small user avatars at the top-left of each poster (opposite side of the episodes-remaining badge).
  • Fallback Letter Avatars: If a user hasn't uploaded a profile image, Jellycheck generates a beautifully styled letter avatar using a unique dynamic gradient generated from the user's name.
