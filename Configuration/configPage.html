How It Works

  • Dynamic Watched Database: The backend reads played state for all users in one fast response.
  • Avatar Overlay: Circular avatars appear on posters showing who has watched each title.
  • Letter Fallback: If a user has no profile image, a gradient letter avatar is shown instead.
  • No page refresh needed: The overlay activates automatically when you visit this page.

Auto-Delete Watched Media

Automatically removes content from Sonarr/Radarr after everyone has watched it.

⚠️ Heads up: Deletions are permanent and remove files from disk. Items are never blacklisted — they can always be re-requested through Overseerr. Favourited items are never deleted.
📺 TV Shows
Enable Auto-Delete for Shows Master toggle. When off, no shows are ever auto-deleted.
Grace Period (days) Days to wait after all users finish a show before deleting it. New episodes reset the clock automatically.
🎬 Movies
Enable Auto-Delete for Movies Separate toggle for movies. Uses its own grace period below.
Movie Grace Period (days) Days to wait after all users watch a movie. Gives time for a potential rewatch.
🛡️ Protection
Skip if Favourited Never delete a title if any user has marked it as a favourite.
